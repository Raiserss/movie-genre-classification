# V2 - Model Comparison and Optimization

## Overview

Version 2 focuses on improving the baseline movie genre classification model by comparing multiple text preprocessing techniques, machine learning algorithms, and hyperparameter configurations.

The objective is to identify the best-performing model while maintaining a systematic experimental workflow.

---

## Dataset

- Dataset: TMDB 5000 Movie Dataset
- Total samples: 4,689
- Number of genres: 15
- Input: Movie overview
- Target: Primary movie genre

## Experiments

| No  | Experiment              | Purpose                                |
| --- | ----------------------- | -------------------------------------- |
| 1   | Baseline                | TF-IDF + Logistic Regression           |
| 2   | Stopword Removal        | Remove common English stopwords        |
| 3   | Unigram + Bigram        | Capture phrase-level features          |
| 4   | Balanced Class Weight   | Handle class imbalance                 |
| 5   | LinearSVC               | Compare with another linear classifier |
| 6   | Multinomial Naive Bayes | Compare probabilistic classifier       |
| 7   | TF-IDF max_features     | Tune vocabulary size                   |
| 8   | Logistic Regression (C) | Tune regularization strength           |
| 9   | Balanced + C=10         | Combine the best configuration         |

---

## Final Model

**Pipeline**

- TF-IDF Vectorizer
  - max_features = 5000
  - stop_words = "english"
  - ngram_range = (1,2)

- Logistic Regression
  - C = 10
  - class_weight = "balanced"

---

## Final Performance

| Metric      |  Score |
| ----------- | -----: |
| Accuracy    | 40.19% |
| Macro F1    | 0.2196 |
| Weighted F1 | 0.3909 |

---

## Key Findings

- Stopword removal slightly improved the baseline model.
- Using unigram and bigram features provided a small performance gain.
- Class balancing significantly improved Macro F1-score.
- Hyperparameter tuning further improved overall performance.
- Combining balanced learning with optimized regularization produced the best overall model.
